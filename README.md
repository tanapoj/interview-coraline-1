# Usage

###  Create Database

```code
cd server
npm install
npx prisma migrate deploy
npx prisma generate
```

### Start Server and Frontend
```code
docker-compose up
```
then, redirect to
```
localhost:4000
```

> Note that: server run on localhost:3000
